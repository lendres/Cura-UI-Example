Introduction

This help file uses various types of styles to communicate meaning. The styles are listed below, along with a short explanation.

Styles

Attribute
The name of an XML attribute.
Definition
A definition is available for this item. Click the item to display the definition. The item can also be found in the glossary.
dialog box/control name
The name of a dialog box, control, or form on the GUI.
Element
The name of an XML element.
.File Extension
A file type or extension.
File Name.ext
The name of a file.
Hyperlink
A link to another page or information.
Important Information
Important or critical notes. The user should pay special attention to this text.
Switch
Command line argument/switch.
Switch or Argument Data
Switch or argument data. For example, some command line arguments/switches require additional data such as the path of a file.
Warning
Critical information. Not following the instructions indicated by this style could result in serious errors or data loss.
