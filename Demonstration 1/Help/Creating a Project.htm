Introduction

Creating a complete project requires the following steps:

  1. Create a new project.
  2. Importing well data, one well at a time.
  3. Enter Formations and Sections.
  4. Specify the project starting and ending depths.

Walk Through

The following steps are used to create and setup a project.

1. Create a New Project

Regardless of whether a project is currently open or no project is loaded, a new project can be started by pressing the New Project button (). Creating a new project will also create default, empty plots and charts.

2. Add Wells

After a new project has been created, wells can be added using the Add Well button (). See Wells for general information about wells. On that page you can also find links to information for getting raw data for wells and importing the data into SurePath.

There is no built in limit to the number of wells that can be added. Rather the limitations of the hardware and the required software performance will determine the practical limit.

3. Add Formations and Sections

Formation and Section data can be entered anytime after the project has been created. To enter data use the Edit Formations () and Edit Sections () buttons for Formations and Sections, respectively. See Formations and Sections Entry for information about entering data.

4. Specify the Project Starting and Ending Depths

The starting and ending depths for a project have two modes: automatic and manual. For a new project the depths are automatically calculated.

To switch to manually entering the start or end depth, click the Edit Starting and Ending Depths button (). See Project Starting and Ending Depths for information about entering starting and ending depths. See Project Segmentation for information about how the starting and ending depths relate to the project intervals and well intervals.
