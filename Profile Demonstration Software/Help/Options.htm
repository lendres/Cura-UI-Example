Form

The image below shows the options form. Refer to the table below for information about the controls.

Column Description
Input Map Controls for specifying the file used to map the columns of input files to internal values. The location of the file is specified here. The file must be edited manually.
Sort Wells Specifies if the wells should be ordered. If this box is checked, when a new well is inserted into the project, the wells will be sorted alphabetically, by name. If this box is not checked, the wells are ordered in the project in the same order that they were added to the project.
Recent Files Menu Sets the number of files shown in the Recent Files Menu.
Default Text The text that is filled in the Well Comparison Table for Void Well Segments (no data available or outside of the starting and ending depths). See Project Starting and Ending Depths and Project Segmentation for more information about what Void Well Segments are and when they occur. See the section Default Text Example for an example of how text entered in this box effects the Well Comparison Table.
Advanced Selection Controls how items are selected in list boxes. If checked, then the shift and control keys can be used to select consecutive files and individual files, respectively. This is similar to how Windows' File Explorer works. If unchecked, each item must be clicked independently. To de-select them, click them again.
